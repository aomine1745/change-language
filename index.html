Why Discord is Sticking with React Native

Being one of the first apps built with React Native, we were excited to share our first year journey using React Native back in 2016.

Looking back at the past three years, React Native has proven to be extremely successful at Discord and helped drive our iOS user adoption from zero to millions!

More specifically, React Native has allowed us to reap the benefits of quickly leveraging reusable code across platforms, as well as develop a small and mighty team.

Meanwhile, we’ve learned to adapt to its inevitable pain points without sacrificing overall productivity.

Por qué Discord se está quedando con React Native

Siendo una de las primeras aplicaciones construidas con React Native, estábamos emocionados de compartir nuestro primer viaje del año usando React Native en 2016.

¡Mirando hacia atrás en los últimos tres años, React Native ha demostrado ser extremadamente exitoso en Discord y ayudó a impulsar la adopción de nuestros usuarios de iOS de cero a millones!

Más específicamente, React Native nos ha permitido cosechar los beneficios de aprovechar rápidamente el código reutilizable en todas las plataformas, así como desarrollar un equipo pequeño y poderoso.

Mientras tanto, hemos aprendido a adaptarnos a sus inevitables puntos débiles sin sacrificar la productividad general.
